# Joseph Nimneh Portfolio

A modern, professional portfolio built with Next.js, TypeScript, and TailwindCSS.

## Features

- **Next.js 14**: Fast, optimized, and SEO-friendly.
- **TypeScript**: Type-safe development.
- **TailwindCSS**: Sleek, modern styling with a dark-mode first approach.
- **Framer Motion**: Smooth micro-animations and transitions.
- **Mobile Responsive**: Perfect layout on all devices.
- **SEO Optimized**: Pre-configured with `next-seo`.

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repository (if applicable) or download the files.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/pages`: Website pages.
- `src/components`: Reusable UI components.
- `src/data`: Data seed files for projects, skills, etc.
- `src/styles`: Global styles and Tailwind configuration.

## Deployment

This project is ready to be deployed on **Vercel**. Simply connect your GitHub repository and it will auto-deploy.
